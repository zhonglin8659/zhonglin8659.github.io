    Python环境安装

    兜里有糖 2024/2/1 Python

    # Python环境安装

    # 下载python

    访问 Python官网 (opens new window) 下载exe安装包

    # 安装python

    双击打开下载的安装包（以windows系统为例）

    勾选底部的Use admin privileges when installing py.exe 与 Add python.exe to PATH

    并选择 Customize installation 安装方式

    继续Next下一步

    选择一个比较友好的目录，利于我们方便找到python程序路径

    开始安装

    安装完毕

    选择点击 Disable path length limit

    # 验证python

    win+R 打开命令行工具输入 python

    再次输入以下内容并回车

    print(“Hello World!!!”)
    1

    到此python环境安装完毕
