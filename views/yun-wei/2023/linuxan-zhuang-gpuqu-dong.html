    Linux安装GPU驱动

    兜里有糖 2023/12/25 Linux

    这里以ubuntu20.04.6-desktop最小安装纯净系统作为演示 显卡为：NVIDIA Quadro M2000M

    # 禁用NOUVEAU开源驱动

    备份linux黑名单文件

    cp /etc/modprobe.d/blacklist.conf /etc/modprobe.d/blacklistbackup.conf
    1

    修改linux黑名单文件

    vim /etc/modprobe.d/blacklist.conf
    1

    在最后一行添加如下内容：

    blacklist nouveau
options nouveau modeset=0
    1
    2

    # 更新使其生效后重启设备

    update-initramfs -u
reboot
    1
    2

    # 查看是否禁用成功

    lsmod | grep nouveau
    1

    没有输出内容即可

    # 下载NVIDIA驱动

    https://www.nvidia.cn/Download/index.aspx?lang=cn (opens new window)

    # 安装GCC编译环境

    apt install gcc
    1

    # 安装MAKE编译环境

    apt install make
    1

    # 安装可执行驱动文件

    chmod +x NVIDIA-Linux-x86_64-535.146.02.run
./NVIDIA-Linux-x86_64-535.146.02.run
    1
    2

    # 安装完成后重启设备

    reboot
    1

    # 检测是否安装成功

    nvidia-smi
    1

    # 安装NVIDIA工具

    apt install nvidia-settings
apt install nvidia-prime
    1
    2

    # 切换独立显卡

    打开NVIDIA X Server Settings点击PRIME Profiles切换到NVIDIA(Performance Mode)选项后重启设备

    # 添加调用环境

    部分系统与软件无法调用独立显卡，请执行以下命令或者加入用户变量中。

    export VK_ICD_FILENAMES="/usr/share/vulkan/icd.d/nvidia_icd.json"
    1

    # 备注

    # linux显卡测试软件glmark2

    # 安装

    apt install glmark2
    1

    # 使用

    glmark2
    1

    # 双系统设置windows为默认

    常规windows10+ubuntu20双系统下开机默认四个选项，顺序代号分别为0、1、2、3

    vim /boot/grup/grup.cfg
    1

    将 set default = 0 改为 set default = 2 开机默认为windows 有8秒选择时间
